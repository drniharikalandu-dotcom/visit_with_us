# Visit With Us Predictor

## 📌 Project Overview
A machine learning pipeline to predict tourism purchase likelihood based on demographic and travel history data.  
Deployed via **GitHub Actions** (CI/CD) and **Streamlit on Hugging Face Spaces**.

---

## 🎯 Objectives
- Automate data preparation and model training.
- Deploy prediction app using Streamlit.
- Ensure reproducible workflows with GitHub Actions.
- Provide evaluator‑friendly documentation and screenshots.
